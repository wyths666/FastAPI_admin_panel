# 🎰 Telegram Bot Platform

#### Платформа из двух Telegram-ботов с веб-админкой для управления заявками, выплатами и поддержкой пользователей.

### 📌 Архитектура проекта

#### Проект состоит из:
- bot_lottery — бот для проведения лотереи
- bot_buyback — бот для обработки выкупов
- admin_api — веб-админка на FastAPI
- MongoDB — отдельная база данных для каждого бота
- Каждый бот изолирован и работает со своей MongoDB.

### ⚙️ Основной функционал
#### 🤖 Telegram-боты
- Регистрация пользователей и сбор реквизитов
- Создание и обработка заявок
- Анонимные чаты по схеме:
- - пользователь → бот → админ → бот → пользователь

- Поддержка статусов заявок
- Интеграция с системой выплат

#### 💳 Управление выплатами
- Автоматическая выплата при подтверждении заявки
- Возможность ручного подтверждения выплаты
- Контроль статусов выплат
- Логирование операций

#### 🖥 Веб-админка (FastAPI)
- Авторизация администраторов
- Управление заявками
- Управление пользователями
- Подтверждение и отклонение заявок
- Ручное управление выплатами
- Просмотр истории операций 
- Интерфейс технической поддержки
